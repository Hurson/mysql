版本：2.0 final  (基线版本：jqGrid 3.5.3)

已经修改部分如下：
添加 tableWidth 变量，解决列数太多导致数据显示过于拥挤的问题，当 tableWidth 大于width的时候就会出现横向的滑动条
添加 widthref 变量，为数组类型，在使用autowidth的时候grid宽度的参照，使用第一个宽度大于0的参照宽度
添加 reloadGridWidthByRef 方法，该方法将遍历 widthref 中的参照物体的宽度，拿到第一个大于0的宽度，然后以此为grid的宽度，重新设置grid的宽度
在 addJSONData 方法最后调用  reloadGridWidthByRef 方法，当数据高度大于grid高度时，重新设置gird的宽度，也就是在右侧添加滑动条的空间

问题建议：
  1、grid第一列的数据如果出现空的时候，选中这一行的时候回报出JS异常
     解决办法：添加第一列为隐藏数据，测数据不能为空
  2、如果第一列数据出现重复的时候，选中其中一行数据，其他第一列数据相同的行也会被选中
     解决办法：添加第一列为隐藏数据，此数据必须永远不会重复

3.0版本建议修改：
   当横向滑动的时候，rownumber 和 multiselect 列最好不跟着滑动，只是后面的数据滑动(UI上要考虑)
   添加一个新的外部方法，批量选中或者取消选中多行数据   （非必须）
   IE6下的使用情况需要测试  （必须）
   firefox 下需要详细测试  （非必须）
   分页的时候，点击到最后一页，再点击到其他页，存在宽度计算错误问题 （超级BUG）
